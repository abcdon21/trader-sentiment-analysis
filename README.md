# trader-sentiment-analysis

# Trader Performance vs Market Sentiment Analysis

## Overview

This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance using historical trading data from Hyperliquid.

The goal is to understand how market psychology influences trading behavior and profitability.

---

## Datasets

1. Historical Trader Data
2. Bitcoin Fear & Greed Index

---

## Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Historical Trader Analysis
5. Fear & Greed Analysis
6. Dataset Merging
7. Sentiment vs Trader Performance Analysis
8. Insights & Conclusion

---

## Key Analyses

- Most traded cryptocurrencies
- Buy vs Sell analysis
- Closed PnL distribution
- Correlation analysis
- Fear & Greed sentiment distribution
- Market sentiment vs trader profitability
- Profit distribution across market sentiments

---

## Results

The analysis highlights how market sentiment affects trading activity and trader performance, providing insights that can support better trading decisions.
